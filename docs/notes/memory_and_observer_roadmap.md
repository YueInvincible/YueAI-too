# Memory And Observer Roadmap

## Muc dich

Giu lai huong memory/observer tu handoff cu, tach khoi handoff tong.

## Dinh huong can giu

- Neu co memory thi can tach ro:
  - working memory;
  - episodic memory;
  - semantic/profile memory.
- Observer khong nen mac dinh thanh pipeline vision nang luon chay.
- Uu tien observer nhieu tang:
  - tang nhe: signal, window/app metadata, event hooks;
  - tang vua: screenshot/OCR theo su kien;
  - tang nang: vision on-demand.

## Rang buoc

- Privacy va permissions phai di truoc.
- Khong luu du lieu nhay cam mot cach mo ho.
- Neu dung SQLite hay store cuc bo khac thi phai quy dinh retention, indexing, retrieval.

## Trang thai

- Chua thay implementation hoan chinh trong repo.
- Day la roadmap/huong thiet ke, khong phai phan da done.

## Viec agent sau nen lam

1. Audit xem repo da co schema/store/memory helper nao chua.
2. Neu co, note file va muc do hoan thanh vao day.
3. Neu chua co, de xuat schema + retrieval strategy toi thieu roi moi code.
